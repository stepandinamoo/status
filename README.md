#  Статус-Сервер

---

##  Что делает код

Веб-сервер с графическим интерфейсом, который позволяет:
- Включать/выключать статус (ON/OFF) с цветовой индикацией
- Выполнять 3 предустановленные команды
- Получать обратную связь после выполнения
- Управлять всем через URL-параметры или кнопки на сайте

---

##  Как запустить
### Требования
- Python 3.x
- Flask

```bash
# Установить Flask (если не установлен)
pip install flask

# Запустить сервер
python app.py
```
После запуска приложение доступоно по адресу `http://localhost:80`

---

##  Использование

После запуска сервера откройте [`http://localhost:80`](http://localhost:80)

На главной странице доступны:

### Управление статусом
- Нажмите **"Включить"** - статус станет ON (зеленый)
- Нажмите **"Выключить"** - статус станет OFF (красный)
- Статус отображается в центре страницы

### Выполнение команд
- **"Команда 1"** - выполняется первая команда
- **"Команда 2"** - выполняется вторая команда
- **"Команда 3"** - выполняется третья команда

После выполнения команды появится сообщение: *"Операция выполнена!"*

### Управление через адресную строку
Можно управлять сервером прямо из адресной строки браузера:

| Действие | URL |
|----------|-----|
| Включить | `/?status=on` |
| Выключить | `/?status=off` |
| Команда 1 | `/?command=1` |
| Команда 2 | `/?command=2` |
| Команда 3 | `/?command=3` |

Пример: если в адресной строке написать `http://localhost:80/?status=on`, статус включится.

### Логирование
Все действия отображаются в консоли, где запущен сервер:
- `Статус изменен на: ON`
- `Выполнена команда 1`
